The form system now supports **multiple subforms**. This enhancement allows dynamic subform generation based on user input.

#### ðŸ§ª How to test it

1. **Select a main form** and begin entering data.
2. When you input specific values, a **popup** may appear indicating that a related subform must be filled.
3. If the main form has **multiple subforms**, additional popups will appear for each triggered condition.
4. Upon submitting the main form, any triggered subforms will be automatically saved as **drafts**.

#### Guest User Experience

- If a subform is assigned to a **guest user**, they will complete it like a regular form without seeing the draft label.

#### Reviewing Subforms

After daily submissions are completed:

1. Go to the **History** screen.
2. Click **View Details** for any main form entry.
3. All associated subforms will be listed below.
4. Each subform will have a **View** button. Click it to see the submitted data.
5. If multiple subforms exist, each will be listed separately with its own View button.
