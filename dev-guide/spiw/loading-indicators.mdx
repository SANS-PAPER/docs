Use this page when implementing or verifying loading states across Sans Paper ID Web. Each action should show the correct indicator component and copy.

## Loading components (shared)

| Component | What it shows | Used for |
| --- | --- | --- |
| `LogoLoader` | Spinning IDlogo.jpg (+ optional label) | Full-page / inline branded loading |
| `AppSpin` | Ant Design Spin with logo indicator | Overlays, sections, modals |
| `FormLoader` | LogoLoader large + "Loading..." | Page/section data fetch |
| `Loader` | Full-screen centered LogoLoader | Auth session bootstrap |
| `getLogoLoaderIndicator()` | Logo inside Ant Design spin | Global default + specific buttons |
| Ant Design `Button loading={…}` | Logo spinner on button (via ConfigProvider in root layout) | Submit / save / action buttons |
| Text only | "Loading...", "Generating...", etc. | Some sub-sections (no logo) |

### Shared components QA checklist

- [ ] `LogoLoader` — spinning logo with optional label
- [ ] `AppSpin` — Ant Design Spin with logo indicator
- [ ] `FormLoader` — large logo + "Loading..."
- [ ] `Loader` — full-screen centered logo
- [ ] `getLogoLoaderIndicator()` — logo inside Ant Design spin
- [ ] Ant Design `Button loading` — logo spinner via `ConfigProvider`
- [ ] Text-only loaders — no logo, correct copy shown

## Global / app-wide

| Action / trigger | Indicator |
| --- | --- |
| Auth0 session resolving (not logged in) | `Loader` (full-screen LogoLoader) |
| Post-login account setup (AuthWrapper) | Full-screen overlay + `AppSpin` — "Setting up your account…" |
| Any Ant Design Spin / button loading app-wide | IDlogo.jpg via `ConfigProvider` in `layout.tsx` |

### Global QA checklist

- [ ] Auth0 session resolving — `Loader` (full-screen LogoLoader)
- [ ] Post-login account setup — overlay + `AppSpin` ("Setting up your account…")
- [ ] App-wide Ant Design Spin / button loading — IDlogo.jpg via `ConfigProvider` in `layout.tsx`
