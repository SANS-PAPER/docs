---
title: 'Staging Server Deployment'
description: 'How to deploy SPF Web to Staging Server'
---
<Note>
  Deployment now are managed automatically from Azure Pipelines.
</Note>

1. Azure Pipelines will auto deploy when pull request to staging is accepted at Github. Please wait around 4 to 5 minutes for the pipelines to finished run.

2. To update .env file, please go to Azure Devops > Pipelines > Library. Add the required data inside the variable group (SpfStagingEnv). Next update the azure-pipelines.yml file to include the new env.
