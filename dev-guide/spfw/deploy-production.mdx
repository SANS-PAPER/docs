---
title: 'Production Server Deployment'
description: 'How to deploy SPF Web to Production Server'
---

<Note>
  Deployment now are managed automatically from Azure Pipelines.
</Note>

1. Azure Pipelines will auto deploy when pull request to master is accepted at Github. Please wait around 4 to 5 minutes for the pipelines to finished run.

2. To update .env file, please go to Azure Devops > Pipelines > Library. Add the required data inside the variable group (SpfProductionEnv). Next update the azure-pipelines-1.yml file to include the new env.
