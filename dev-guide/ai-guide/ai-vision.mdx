## Login on Microsoft Azure AI Vision
- Go to [Vision Studio](https://portal.vision.cognitive.azure.com/)
- Click on 'Login Options'.
- Login as organization.
- Use 'sanspapergroup.onmicrosoft.com' as organization name.

## Using the API

<Note>
  The value `VITE_AZURE_AI_VISION_URL` and `VITE_AZURE_AI_VISION_KEY` are defined at .env file for frontend SPF Web project.
</Note>

- Use this code to call the API

```js
let data = JSON.stringify({
    "url": filename //url location of the image
});
    
let config = {
    headers: {  
        'Content-Type': 'application/json', 
        'Ocp-Apim-Subscription-Key': `${import.meta.env.VITE_AZURE_AI_VISION_KEY}`
    }
};

const result : AxiosResponse = await axios.post(`${import.meta.env.VITE_AZURE_AI_VISION_URL}`, data, config); 
```
