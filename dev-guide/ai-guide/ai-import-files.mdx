---
title: 'Add Documents to Azure AI'
description: 'Guides on upload and generating index on Microsoft Azure AI to be used on Analyze Chat'
---

## Login on Microsoft Azure Portal
- Go to [Microsoft Azure Portal](https://portal.azure.com/).
- Login on new Azure account.

## Uploading documents
- On the dashboard, click the `Storage Accounts` service on the left sidebar.
- Click on storage name `sanspapergroup` .
- Click on dropdown menu `Data storage` .
- Click on `Containers` .
- Click on container name `azumeml` .
- Click 'upload' to upload files into the folder.

## Generating Index for Microsoft Azure AI
- On the dashboard, click the `Resource groups` service on the left sidebar. 
- Click on resource `AI-search` .
- Click on `sanspaper-ai-search` which is a search service.
- Click on `Import Data (new)` .
- Choose data source as `Azure Blob Storage` .
- Choose scenario as `RAG` .
- Select storage account `sanspapergroup` .
- Select blob container `azureml` . Then click Next.
- Select Azure OpenAI service as `azure-openai-search` .
- Select model deployment as `text-embedding-ada-002` .
- Tick the acknowledge button and click Next.
- Click next on `vectorize image` page.
- Click next on `advanced setting` page.
- On page `review and create`, update the object name or leave as default. Click create to start creating the index. This name is the index name to be used in the Azure AI chat.
- After finished creating index, the index will be created at the page `sanspaper-ai-search`. 
- To confirm the index name, click the dropdown `Search Management` at the left sidebar. Then click `Indexes` to see the newly created index.
- This index name is now the search index to be used in Azure Chat search index.

## Update the Index Name in Applications
- Copy the newly created search index name and replace the index name inside the respective project .env file.

```env
VITE_AZURE_SEARCH_INDEX=rag-1759226008449 #replace this value with the new index name
```

- Rebuild the project and deploy as usual.
