## Running the developer view locally

To preview the full developer documentation site (including deployment guides, API reference, and AI guides):

1. Copy the contents of `mint_dev.json` into `mint.json`.
2. From the repo root, run:

```bash
mintlify dev
```

3. Open `http://localhost:3000` and use the **Developers Guide** tab in the top navigation.

To preview the public customer site instead, copy `mint_public.json` into `mint.json` before running `mintlify dev`.
