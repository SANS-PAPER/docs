title: 'Local Deployment'
description: 'How to deploy SP Pilot on local machine'
---


## <Icon icon="list-check"  size={30}/> Prerequisite
- Node.js v18.20 or above

## <Icon icon="Github" size={30}/> Clone Github Project
- Download project from Github. [SansPaper-Pilot](https://github.com/SANS-PAPER/SansPaper-Pilot).

## <Icon icon="gears"  size={30}/> Configure Project SP Pilot
- Rename file .env-local to .env
- Run command below.

```sh
yarn install
```

- Make sure the command finished successfully without any error.

## <Icon icon="react" size={30}/> Start the project
- Run command below.

```sh
npm run dev
```

## <Icon icon="eye"  size={30}/> Accessing the project 

- To open the SansPaper Pilot, use this url [http://localhost:3000/](http://localhost:3000/).
